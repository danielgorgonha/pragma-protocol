# One-Pager: Pragma Token (PGM)

Tokenomics — Lesson 01. Decentralized Web3 services marketplace.  
Author: Daniel Gorgonha, Pragma Protocol. February 2026.

## 1. Problem and value proposition

The Web3 services market (development, audits, consulting) operates mainly in ETH or stablecoins, with no native incentive layer that retains value in the ecosystem or rewards long-term participants.

PGM addresses this as the medium of exchange and access key for the Pragma marketplace: buyers and providers transact with incentives aligned to platform growth.

## 2. Token architecture

| Attribute   | Value |
|------------|-------|
| Name       | Pragma Token |
| Symbol     | PGM |
| Standard   | ERC-20 (Ethereum, Sepolia) |
| Total supply | 10,000,000 PGM (fixed, no mint after deploy) |
| Decimals   | 18 |
| Type       | Utility token |

## 3. Token functions

**Payment.** All marketplace transactions (hiring, milestones, fees) are settled in PGM. The platform retains 2% for the ecosystem reserve.

**Access.** Holders with ≥ 500 PGM (providers) get reduced fee (1%), featured listings, priority access. Holders with ≥ 200 PGM (buyers) get access to verified providers and priority support.

## 4. Supply and distribution

| Allocation   | %  | Tokens  | Vesting |
|-------------|----|---------|---------|
| Public sale | 40 | 4,000,000 | None |
| Ecosystem   | 30 | 3,000,000 | Gradual by usage |
| Team        | 20 | 2,000,000 | 12m cliff + 24m linear |
| Reserve     | 10 | 1,000,000 | 6m cliff |

## 5. Economic sustainability

- Fixed supply: no further emission.
- Value capture: 2% fee funds ecosystem reserve and rewards.
- Velocity: premium benefits create incentive to hold (lower V in MV=PQ).
- Team vesting: 12-month cliff reduces sell pressure at launch.

## 6. Risks and mitigations

| Risk              | Mitigation |
|-------------------|------------|
| Low adoption      | Public faucet |
| Concentration     | 40% public sale, long team vesting |
| Volatility        | Fixed supply, premium access |
| Regulatory        | Pure utility, no return promise |
