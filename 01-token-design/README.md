# Lesson 01 — Fundamentals and Architecture of Tokens

Tokenomics design for **Pragma Token (PGM)**, a utility token for a decentralized Web3 services marketplace.

## Design summary

- ERC-20, fixed supply 10M PGM
- Functions: payment (medium of exchange) and access (utility gate)
- Distribution: 40% public sale, 30% ecosystem, 20% team, 10% reserve
- Team vesting: 12-month cliff, 24-month linear

The PGM token can later be extended with the BME mechanism from Lesson 02.
